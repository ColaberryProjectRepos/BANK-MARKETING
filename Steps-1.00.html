Step 1: Project Overview and Process flow 1



                                                     




                                                      


Technologies: Python, Jupyter Notebook, Spyder

Data Sources:  UCI Repository

Description: Bank’s Fixed term deposits are very economical. Banks invest that money in other firms and make profits. So, any bank doesn’t want to lose any of their clients.

Aim of this project is to create a Classification model to predict if the client will subscribe a term deposit or not.

Also, to improve business strategies like whom to target and when to target to make the campaigns effective.

The Scope of this project is to visualize the data to get useful insights which will be beneficiary to develop business strategies and to build Classification models like logistic Regression and k-NN classifier and validate them to find the best fit for the data


Process Flow:

                          

  • The process flow involves the following steps
  • Importing the data into Jupyter Notebook
  • Data Pre-processing like checking for the null, values cleaning the data etc.
  • Exploratory Data Analysis of the data and visualizing the data for insights
  • Models implementation (Logistic regression and KNN classifier)
  • Validating the models using confusion matrix and RoC (Receiver Operating Characteristics) curve to find the best model fit
